RIOLib - Remote/Radio Input-Output Library for wired and wireless communication.

RIOLib stands for Remote/Radio Input-Output Library and it is 
a simple serial communication library for wired and wireless modules.

This is a simple serial communication library for wired and wireless modules.

Folders and modules:
 - riolib            -
 - riolib_recv1      -
 - riolib_send1      -
 - riolibc_recv1      -
 - riolibc_send1      -

This was developed for and tested on the Tinusaur - a ATtiny85 microcontroller board.
