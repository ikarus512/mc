# Banana Pi BPI-M2 ZERO

  www.banana-pi.org

# Key features

CPU: Allwinner H2+, Quad-core Cortex-A7.
512MB DDR 3 SDRAM.
WiFi (AP6212) & Bluetooth onboard.
Mini HDMI.
40 Pin Raspberry Pi-compatible GPIO, including UART, SPI, I2C, etc.
